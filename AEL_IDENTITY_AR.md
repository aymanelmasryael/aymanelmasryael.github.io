## نظام التصميم

| العنصر | المعيار |
|--------|---------|
| **السمة** | الوضع الليلي (Dark mode) |
| **الأسلوب** | Glassmorphism، زوايا سيبرانية، لمسات monospace |
| **الإطار** | HTML5 + CSS3 + vanilla JS (مضمن للصفحات) |
| **الأيقونات** | SVG sprite sheet (Font Awesome Free 6.4.0) |
| **الخط** | Inter (أساسي)، system-ui كاحتياطي |
| **الشعار** | نمط شبكي AEL مخصص — مصفوفة مكعبات زرقاء |
| **التوقيع** | "أيمن المصري \| AEL Digital Studio" |
| **التذييل** | © 2026 أيمن المصري — AEL Digital Studio |
| **الأداء** | هدف Lighthouse 100/100 |
| **النشر** | GitHub Pages مع `.nojekyll` |

**اتفاقيات CSS:**
- البادئة: `ael-` (مثل: `.ael-glass-card`، `.ael-btn-primary`)
- المسافات: خصائص CSS مخصصة (`--ael-space-*`)
- الزوايا: `--ael-radius-*` (xs إلى full)
- الانتقالات: `--ael-transition-base` (250ms cubic-bezier)
