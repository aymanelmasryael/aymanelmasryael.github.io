## Brand Philosophy — The Three Pillars

### Pillar I — Artist Over Tool

>
> الذكاء الاصطناعي يعطي الأدوات، لكن الفنان هو من يحولها إلى تحف فنية.

  <em>— Ayman Elmasry</em>

### Pillar II — Substance Over Noise

> "Fame does not mean quality, and noise does not mean efficiency. True quality reveals itself in silence, while noise disappears when it truly matters."
>
> الشهرة لا تعني الجودة، والضجيج لا يعني الكفاءة. الجودة الحقيقية تظهر في صمت، بينما يختفي الضجيج عندما يحين وقت الإنجاز الحقيقي.

<div align="right">
  <em>— Ayman Elmasry</em>
</div>

---

### Pillar III — Engine Over Output

> "I don't design logos. I design *logo generators*. I don't build images. I build *engines that produce images*."
>
> أنا لا أصمم الشعارات، بل أصمم *مولدات الشعارات*. ولا أبني الصور، بل أبني *محركات تُنتج الصور*.

<div align="right">
  <em>— Ayman Elmasry</em>
</div>

## Design System

| Element | Standard |
|---------|----------|
| **Theme** | Dark mode |
| **Style** | Glassmorphism, cyber corners, monospace accents |
| **Framework** | HTML5 + CSS3 + vanilla JS (inline for Pages) |
| **Icons** | SVG sprite sheet (Font Awesome Free 6.4.0) |
| **Font** | Inter (primary), system-ui fallback |
| **Logo** | Custom AEL grid pattern — blue block matrix |
| **Signature** | "Ayman Elmasry \| AEL Digital Studio" |
| **Footer** | © 2026 Ayman Elmasry — AEL Digital Studio |
| **Performance** | Lighthouse 100/100 target |
| **Deployment** | GitHub Pages with `.nojekyll` |

**CSS Conventions:**
- Prefix: `ael-` (e.g., `.ael-glass-card`, `.ael-btn-primary`)
- Spacing: CSS custom properties (`--ael-space-*`)
- Radius: `--ael-radius-*` (xs through full)
- Transitions: `--ael-transition-base` (250ms cubic-bezier)
